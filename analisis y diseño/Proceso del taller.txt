-> El cliente lleva su motor electrico al taller para revisión por un mal funcionamiento

-> El motor se recibe y se realiza una revisión basica para encontrar un posible daño 

-> si el daño es sencillo se notifica al cliente el costo de la refacción y que
se puede realizar la refacción de inmediato.

-> si el cliente decide realizar la refacción se realizan los respectivos cambios, se genera la factura y finaliza el proceso,
de lo contrario, el cliente se lleva su motor y finaliza el proceso. 

-> si el daño es complejo de identificar se notifica que se deber realizar una revisión completa.
Seguido de esto el cliente debe registrar su nombre completo y numero telefonico

 
-> Realizar la revisión completa. luego de identificar el daño se le notifica al cliente el estado del motor 
y el costo de la refacción.
Si el cliente autoriza la reparación, se continúa con el proceso. De lo contrario, se emite una factura por la revisión completa y
se realiza el cobro correspondiente, el cliente retira su motor y decide el metodo de pago (efectivo o tranferencia), 
dando por finalizado el proceso

-> El proceso continua realizando un cambio de rodamientos o rebobinado o ambas operaciones según corresponda.

-> luego de realizar la refacción se notifica al cliente el proceso realizado y que puede pasar por su motor.

-> Si cliente pasa por su motor y recibe la factura, procede a realizar el pago y decide el metodo de pago (efectivo o tranferencia),
para dar fin al proceso, de lo contrario, sino recoge el motor los 15 días se vuelve a notificar que deber pasar por el motor.
